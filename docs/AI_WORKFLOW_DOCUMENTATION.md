# AI Workflow

## *Which AI was used and why?*
I primarily work with ChatGPT and Claude, so I used both tools for development.

## *What tasks were solved with the help of AI?*
AI was used to assist with creating UI components, implementing drag-and-drop functionality, defining API endpoints, and fixing bugs or improving the generated code.

*Examples:*
- Generating the initial project structure and core functionality
- Writing API endpoints for task CRUD operations
- Refactoring code to improve readability and structure
- Fixing TypeScript errors related to status types and route parameters

## *What new aproaches or techniques did you find via AI? How it helped to facilitate development?*
Most API-related techniques were already familiar to me. However, drag-and-drop functionality was new.
The application currently uses native HTML5 drag and drop, but during development I learned about dnd-kit, and I plan to experiment with it in future projects.

## *Were there any difficulties and how you solved them?*

I identified both strengths and weaknesses in each AI tool:

### **Claude:**  
Pros:
- Better error handling for API routes
- More polished UI (Add task button, close button for modal)

Cons:
- Did not include PUT/DELETE endpoints
- Initially overwrote the task list instead of appending new tasks to the existing array


### **ChatGPT**  
Pros:
- Stronger Typescript typing
- Included PUT requests
- Created reusable helper functions

Cons:
- No Add task button initially
- Initially missed DELETE endpoint
- Lacked proper error handling for API endpoints
- generated PUT request for /task instead of /task/:id

### **Reflections:**  
The UI generated by Claude was visually better, so I used it as a starting point. However, I had to manually refactor the code to export shared constants and types to improve maintainability.
I  tested the API endpoints and added proper validation for POST and PUT requests. During this process, the build initially failed due to incorrect typing of route parameters in PUT and DELETE handlers. After correcting the parameter types, the application built and ran successfully.

Overall, using both ChatGPT and Claude allowed me to combine better UI generation with stronger typing and backend logic, resulting in a more complete and stable application.
